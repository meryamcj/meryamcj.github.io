Colombia's President Word Cloud

In this project I will be using some web scraping tools for python to have a simple sight of the most common words used on two different speechs from the President of Colombia found on the government official page for transcriptions and press webpages. After retrieving our data using BeautifulSoup, I will be representing the top 10 most common words on each one using Matplotlib. The idea is to compare the difference between them within three years apart.

Video game sales analysis and prediction with Pandas Profiling

In this project I am using a dataset from Kaggle to build an analysis of the sales of video games with some help from Pandas Profiling which is an open source Python module with which we can quickly do an exploratory data analysis. It help us to visualize and understand the distribution of each variable.
